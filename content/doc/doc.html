Allgemeine Beschreibung:

Das Projekt "Forum" ist eine Client-Server-Anwendung zur Darstellung einzelner Webseiten und Formulare, die per Template-Engine mako erzeugt werden.
Nutzung des Frameworks cherrypy für den Server. Die Präsentation des Projektes erfolgt per CSS.
Zum Speichern der Daten des Forums (Themen, Diskussionen und Beiträge) und der Konten der Benutzer werden JSON-Dateien genutzt.
Der Austausch von Formulardaten zwischen Client und Server erfolgt per AJAX.

API:

  • /login
    • required parameters - (user, password)
  • /logout
  • /create_user
    • required parameters - (alias, role, name, password)
  • /update_user
    • required parameters - (alias, role, name)
    • optional parameters - (password)
  • /delete_user
    • required parameters - (alias)
  • /create_theme
    • required parameters - (name)
  • /create_discussion
    • required parameters - (theme, title, article_title, content_title)
  • /update_discussion
    • required parameters - (theme, discussion, title)
  • /delete_discussion
    • required parameters - (theme, discussion)
  • /create_article
    • required parameters - (theme, discussion, title, content)
  • /update_article
    • required parameters - (theme, discussion, article, title, content)
  • /delete_article
    • required parameters - (theme, discussion, article)
